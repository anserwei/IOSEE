## Versioning Policy

IOSEE follows [Semantic Versioning 2.0.0](https://semver.org/):

- **MAJOR** — incompatible public API changes.
- **MINOR** — backward-compatible new capability (e.g. new modes,
  new physical kernels, new configuration options).
- **PATCH** — backward-compatible bug fixes and documentation.

The `0.8.x` and `0.9.x` series constituted the public beta used during
internal validation against in-situ and satellite reference data. Release
**1.0.0** marks the first stable version with a frozen public API and the
full physics pipeline validated across the 10 – 5000 cm⁻¹ infrared range.
